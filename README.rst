A Django app to track ping pong games. For Python 2.7/3.2+ and Django 1.8+.

Documentation
-------------

The full documentation is at https://django-pong.readthedocs.org.

Quickstart
----------

Install django-pong::

    pip install django-pong

Then use it in a project::

    import pong
